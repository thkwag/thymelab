<h1 align="center">ThymeLab - Thymeleaf Templates, Instantly Previewed</h1>

ThymeLab is a development tool that enables designers and developers to create and test Thymeleaf templates without setting up a backend system. It consists of two main modules:

## Key Features

### Easy Template Development
- Create and test Thymeleaf templates without backend implementation
- Real-time hot reload for instant preview
- Simple JSON-based template variable management
- Layout system and fragment support

### User-Friendly Interface
- One-click installation with bundled JVM
- No manual Java environment setup required
- Visual configuration through desktop application
- Real-time log monitoring

## Modules

### [ThymeLab Processor](modules/processor/README.md)
- Thymeleaf template processing engine
- Hot reload functionality
- JSON data management
- Static resource serving
- Layout system support

### [ThymeLab Launcher](modules/launcher/README.md)
- Desktop application for managing processor
- Easy directory configuration
- Server control interface
- Real-time log display
- System tray integration

## Quick Start

1. **Installation**
   - Download the installer for your platform (Windows/macOS)
   - Run the installer
   - Launch ThymeLab from your applications menu

2. **Configuration**
   - Set up directories for templates, static resources, and data
   - Configure server port if needed (default: 8080)
   - Adjust log level and display settings

3. **Development**
   - Start the server
   - Open `http://localhost:8080` in your browser
   - Edit templates and see changes instantly

## System Requirements

- Windows 10/11 or macOS 10.14+
- No additional Java installation required
